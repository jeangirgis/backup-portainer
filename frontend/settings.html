☁️ Google Drive Setup

If you want to use Google Drive as your storage backend (STORAGE_BACKEND=gdrive), you must set up a Google Service Account.

  1. Go to the Google Cloud Console.
  2. Create a Project and enable the Google Drive API.
  3. Navigate to IAM & Admin > Service Accounts and create a new Service Account.
  4. Create a JSON Key for that Service Account and save it as credentials.json.
  5. Create a folder in your personal Google Drive and Share it with the Service Account's email address (give it Editor access).
  6. Extract the Folder ID from the URL (the long string of characters after folders/).
  7. Mount the credentials.json file into your docker container and set GDRIVE_FOLDER_ID in your .env file.
